# Credit Card Fraud Detection Dashboard

## Overview

This Power BI dashboard provides a clear and interactive view of credit card transactions to help identify and analyze fraudulent activity. It supports quick insights into fraud patterns and key risk indicators.

## Key Features

* Fraud vs Non-Fraud transaction overview
* Transaction and fraud trends over time
* Key fraud KPIs and risk indicators
* Interactive filters for deeper analysis

## Key Metrics

* Total Transactions
* Fraudulent Transactions
* Fraud Rate (%)
* Total Fraud Amount

## Tools Used

* Power BI Desktop
* DAX for calculated measures

## Usage

1. Open the `.pbix` file in Power BI Desktop
2. Refresh data if required
3. Use filters and visuals to explore fraud patterns

## Notes

* Data may be anonymized and used for analysis/demo purposes
* Dashboard supports analysis, not real-time fraud detection
